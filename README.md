A cute, interactive Valentine's proposal app built with React, TypeScript, and Vite.

## Features

- **Rose/Pink Theme**: A romantic design with custom styling.
- **Interactive "No" Button**: The button playfully escapes your cursor or touch, making it impossible to say no! ðŸ˜ˆ
- **Celebration Mode**: Clicking "Yes" triggers a confetti explosion and success message.

## Tech Stack

- React
- TypeScript
- Vite
- Framer Motion (for animations)
- Canvas Confetti (for celebration)

## Getting Started

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```

## Deployment

This project includes a GitHub Actions workflow to automatically deploy to GitHub Pages. simply push to the `main` branch.
